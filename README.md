# Django 
### Создайте виртуальное окружение(venv) 
   
``` python -m venv venv ```
### активируйте venv

``` ./venv/Scripts/activate ```

### установите зависимости 

``` pip install -r requirements/test.txt ```

### запустите проект 

``` python lyceum/manage.py runserver ``` 

или 

``` cd lyceum```

``` python manage.py runserver```
